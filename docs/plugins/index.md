## 更多插件

- **Calendar** - 日历插件：事件聚合，多视图展示
- **Day** - 纪念日插件：倒计时/正计时管理
- **Todo** - 任务插件：子任务，优先级，日期范围
- **Tracker** - 目标追踪插件：量化目标，数据记录
- **Contact** - 联系人插件：信息管理，互动历史
- **Timer** - 计时器插件：多种计时方式
- **Store** - 物品兑换插件：积分系统
- **Nodes** - 节点插件：笔记本树结构
- **Calendar Album** - 日记相册插件：照片标签，日期记录
- **Habits** - 习惯管理插件：技能关联，一万小时定律
- **Database** - 自定义数据库插件：灵活字段定义

## 插件开发

Memento 支持自定义插件开发，您可以：

1. **启用/禁用**：根据需要启用或禁用特定插件
2. **自定义配置**：调整插件参数以满足个人需求
3. **数据导入导出**：备份和恢复插件数据
4. **开发新插件**：使用插件开发工具创建专属功能

## 使用建议

- **新手推荐**：从 Chat、Diary、Notes 三个核心插件开始
- **效率提升**：配合 Activity、Todo、Bill 使用
- **生活管理**：结合 Goods、Checkin、Habits 管理日常
- **深度用户**：探索 Database、Tracker 等高级功能
